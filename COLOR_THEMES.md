## üí° Usage Tips

### For Maximum Scannability
1. Choose themes with ‚≠ê‚≠ê‚≠ê‚≠ê‚≠ê rating
2. Test scan before printing
3. Avoid adding large logos
4. Print at minimum 2cm size

### For Branding
1. Match your brand colors
2. Use custom colors if presets don't match
3. Check contrast warning
4. Test on actual materials

### For Print
1. Use high-contrast themes
2. Test print on actual paper/material
3. Consider lighting conditions
4. Use SVG format for large prints

### For Digital
1. Any theme works well
2. Consider screen brightness
3. Test on different devices
4. PNG format recommended

## üìä Color Psychology

### Red Themes (Cherry, Ruby, Crimson, Fire, Blood, Scarlet)
- **Emotions**: Passion, energy, urgency, love, power
- **Use For**: Sales, food, romance, attention-grabbing
- **Avoid For**: Calming, medical (except blood donation)

### Wine & Burgundy
- **Emotions**: Sophistication, luxury, maturity
- **Use For**: Upscale, vintage, wine-related
- **Avoid For**: Youth-oriented, casual

### Blue (Ocean)
- **Emotions**: Trust, calm, professional, tech
- **Use For**: Corporate, tech, healthcare
- **Avoid For**: Food (suppresses appetite)

### Green (Forest)
- **Emotions**: Nature, health, growth, eco
- **Use For**: Organic, environmental, health
- **Avoid For**: Luxury (unless eco-luxury)

### Purple
- **Emotions**: Luxury, creativity, wisdom
- **Use For**: Beauty, premium, creative
- **Avoid For**: Corporate, traditional

### Black & White (Classic, Dark)
- **Emotions**: Professional, timeless, elegant
- **Use For**: Any professional use
- **Universal**: Works everywhere
